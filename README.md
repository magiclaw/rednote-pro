# RedNote Pro - 小红书笔记排版工具

将 Markdown 内容排版为小红书风格的卡片图片，支持多种样式、封面设置与一键导出。纯前端单文件，无需后端，开箱即用。

---

## 功能概览

- **多种视觉样式**：极光、清新、浪漫、国潮、抹茶、简约等，一键切换
- **封面与正文排版**：自定义封面标题/副标题、字体、字号、布局、刊号显示
- **正文 Markdown**：支持标题、加粗、列表、图片等，实时预览
- **导出**：单张图片或整份 ZIP 打包下载，适合直接发小红书
- **零依赖安装**：脚本与字体均通过 CDN 加载，用浏览器打开 `xshnew.html` 即可使用

---

## 快速开始

### 本地使用

1. 用浏览器直接打开 **`xshnew.html`**（双击或在本地服务器中打开均可）
2. 在左侧边栏：
   - 选择**风格**（极光 / 清新 / 浪漫等）
   - 填写**封面内容**（大标题、副标题、是否显示刊号等）
   - 在**正文内容**中粘贴或输入 Markdown
3. 右侧实时预览卡片效果
4. 点击底部「下载」导出图片或 ZIP

### 在线部署（可选）

- 可部署到 **GitHub Pages**、**Vercel**、**Netlify** 等静态托管，获得一个在线访问链接
- 仓库内仅需包含 `xshnew.html` 及本 README 等文件即可

## 技术说明

- **前端**：HTML + CSS + JavaScript，单文件内联样式与脚本
- **依赖（CDN）**：marked（Markdown 解析）、html2canvas（导出图片）、JSZip、FileSaver.js、Phosphor Icons、思源/霞鹜等字体
- **兼容**：现代浏览器（Chrome、Safari、Edge、Firefox 等）
