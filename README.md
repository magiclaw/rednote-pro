# RedNote Pro - 小红书笔记排版工具

将 Markdown 内容排版为小红书风格的卡片图片，支持多种样式、封面设置与一键导出。

## 使用

1. 用浏览器直接打开 **`xshnew.html`**，或部署到任意静态托管（GitHub Pages、Vercel 等）
2. 在左侧输入/选择内容与样式
3. 预览后使用「下载」或「发布」导出

## 功能

- 多种视觉样式（极光、清新、浪漫、国潮等）
- 封面与正文排版自定义
- 导出为图片或 ZIP
- 依赖均通过 CDN 加载，单文件即可运行

## 技术

- 纯前端：HTML + CSS + JavaScript
- CDN：marked、html2canvas、JSZip、FileSaver.js、Phosphor Icons、思源字体等
